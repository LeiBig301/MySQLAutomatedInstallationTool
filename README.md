# MySQL 自动安装工具使用说明书

## 1. 软件简介

MySQL 自动安装工具是一款专为 Windows 系统设计的 MySQL 数据库一键安装配置工具，旨在简化 MySQL 的安装和配置过程，让用户无需手动执行复杂的安装步骤。

**版本：1.1.12**

## 2. 功能特性

- ✅ 跨平台字体安全（避免乱码）
- ✅ MySQL 版本可搜索下拉框（自动补全）
- ✅ 自动下载解压、环境变量设置
- ✅ 兼容 Win7 - Win11
- ✅ 加载提示与平滑版本更新
- ✅ 管理员权限自动检查
- ✅ 完整的安装流程引导
- ✅ 服务管理功能（删除服务）
- ✅ 详细的日志记录

## 3. 系统要求

- **操作系统**：Windows 7 及以上版本（32位/64位）
- **Java 环境**：JRE 1.8 及以上版本
- **网络连接**：下载 MySQL 时需要互联网连接
- **磁盘空间**：至少 500MB 可用空间
- **管理员权限**：需要以管理员身份运行

## 4. 安装步骤

### 4.1 运行程序

1. 双击 `MySQLAutomatedInstallationTool.jar` 文件
2. 如果没有管理员权限，会自动弹出提示框，点击「是」以管理员身份重新运行
3. 程序启动后，会自动检测 Windows 版本并显示在界面上

### 4.2 配置参数

在主界面中填写以下配置：

| 参数 | 说明 | 示例 |
|------|------|------|
| MySQL 版本 | 选择或搜索要安装的 MySQL 版本 | 8.0.35 |
| 安装路径 | MySQL 安装目录 | C:\mysql |
| 端口 | MySQL 服务端口 | 3306 |
| 服务名 | Windows 服务名称 | MySQL |
| Root 密码 | MySQL root 用户密码（留空表示无密码） | 123456 |

### 4.3 开始安装

1. 点击「开始安装」按钮
2. 程序会自动执行以下步骤：
   - 下载并解压 MySQL（如果本地不存在）
   - 初始化数据库
   - 注册并启动 Windows 服务
   - 设置 root 密码（如果指定）
   - 配置环境变量
3. 安装过程中会显示进度条和详细日志
4. 安装完成后会弹出成功提示

## 5. 使用方法

### 5.1 连接 MySQL

安装完成后，可以通过以下方式连接 MySQL：

1. 打开命令提示符（以管理员身份运行）
2. 输入命令：`mysql -u root -p`
3. 输入设置的密码（如果有）
4. 成功连接后，可以执行 SQL 命令

### 5.2 管理 MySQL 服务

#### 启动服务
- 命令：`net start MySQL`（将 MySQL 替换为您设置的服务名）

#### 停止服务
- 命令：`net stop MySQL`

#### 删除服务
- 在工具界面点击「删除服务」按钮
- 选择要删除的 MySQL 服务
- 可选择是否清理环境变量、配置文件和安装文件

## 6. 常见问题

### 6.1 权限不足
- **问题**：运行程序时提示「需要管理员权限才能运行此程序」
- **解决**：以管理员身份运行程序

### 6.2 端口占用
- **问题**：安装时提示「端口 3306 已被占用」
- **解决**：更换其他端口，或关闭占用该端口的程序

### 6.3 下载失败
- **问题**：下载 MySQL 时失败
- **解决**：检查网络连接，或尝试其他 MySQL 版本

### 6.4 服务启动失败
- **问题**：服务注册成功但启动失败
- **解决**：检查配置文件，确保端口未被占用，或尝试重新安装

### 6.5 环境变量未生效
- **问题**：安装后在命令提示符中无法直接使用 mysql 命令
- **解决**：重启命令提示符，或手动检查系统环境变量

## 9. 注意事项

- 安装过程中请保持网络连接稳定
- 安装完成后，建议重启命令提示符以确保环境变量生效
- 如需远程访问，请手动修改 `my.ini` 中的 `bind-address` 为 `0.0.0.0` 并重启服务
- 定期备份数据库，以防数据丢失
